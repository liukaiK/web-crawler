无障碍功能操作说明

根据中华人民共和国信息产业部发布的《信息无障碍网站设计无障碍技术要求》的相关标准为技术依据，实现了强化级别的无障碍访问。 本网站的无障碍操作概述包含以下内容：

  1. 什么是信息无障碍
  2. 什么是网页无障碍
  3. 网站无障碍功能

什么是信息无障碍？

所谓的信息无障碍是指无论健全人还是残疾人、无论年轻人还是老年人都能够从信息技术中获益，任何人在任何情况下都能平等地、方便地、无障碍地获取信息、利用信息。

什么是网页无障碍？

所谓的网页无障碍是指残疾人、有特殊需求的健全人可以获取网络上的任何信息，为了做到这一点，就要实现网页内容无障碍以及上网使用的辅助软件技术的无障碍。

网站主要实现了以下三个方面的无障碍功能

  1. 全网站页面可用键盘操作，不限于鼠标
  2. 语义化的辅助提示语音功能
  3. 无障碍网站浏览辅助功能

全网站页面可用键盘操作，不限于鼠标

  1. 提供文字大小控制功能；
  2. 提供页面色彩对比功能；
  3. 提供光标功能；
  4. 提供界面控制功能

文字大小控制功能包括？

网页浏览者可根据自己的需求自行调整网页文字的大小，文字放大是将网站当前页面的文字放大；文字缩小是将网站当前页面的文字缩小。

光标的作用是什么？

本站提供的光标是横竖两条红色的基准线，为视力障碍用户校队阅读的位置提供了帮助

界面控制功能包括哪些？

网站浏览者可根据自己的需求自行调整网页界面的大小，界面放大是将网站当前页面显示比例放大，界面放大后网页内容也会随之变大；界面缩小是将网站当前页面显示比例缩小，界面缩小后网页内容也会随之变小。
